## Dataset

This project uses the Massachusetts Buildings and the Inria Aerial Image Labeling Datasets.

The Massachusetts Buildings dataset is publicly available at:
https://www.cs.toronto.edu/~vmnih/data/

or via Kaggle:
https://www.kaggle.com/datasets/balraj98/massachusetts-buildings-dataset

The Inria Aerial Image Labeling Dataset is publicly available at:
https://www.kaggle.com/datasets/sagar100rathod/inria-aerial-image-labeling-dataset

Due to size and licensing constraints, the datasets were not included.
